# moocHelper
智慧职教助手第一版

## 程序原理
- 模拟一次答题，然后查看答案，将答案保存到本地，利用本地的答案答题

## 命令：

- pip install requests -i https://pypi.tuna.tsinghua.edu.cn/simple
- pip install beautifulsoup4 -i https://pypi.tuna.tsinghua.edu.cn/simple
